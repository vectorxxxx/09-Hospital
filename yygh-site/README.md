## 2、scrollIntoView() 是什么？

```javascript
document.getElementById(depcode).scrollIntoView();
```

`scrollIntoView()`：这个方法用于将元素滚动到视口可见区域。它接受一个可选的参数`alignToTop`，表示元素是否应该与视口顶部对齐。如果该参数设置为`true`，则元素将从顶部开始滚动；如果设置为`false`（默认值），则元素将从底部开始滚动。
